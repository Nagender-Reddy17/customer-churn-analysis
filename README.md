# 📊 Customer Churn Analysis

## 🎯 Objective

The goal of this project is to analyze customer churn behavior and identify key factors influencing customer retention. The project focuses on understanding patterns in customer demographics, service usage, and tenure to reduce churn and improve business decisions.

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Seaborn, Matplotlib)
* SQL Server (SSMS)
* Power BI
* Streamlit (for deployment)

## 🧹 Data Cleaning

* Standardized column names
* Handled missing values
* Converted data types
* Encoded churn variable (Yes/No → 1/0)

---

## 📊 Exploratory Data Analysis (EDA)

* Analyzed churn distribution
* Studied impact of tenure, income, and age
* Evaluated service usage (internet, wireless)
* Identified high-risk customer segments

---

## 🗄️ SQL Analysis

Performed KPI calculations and segmentation:

* Churn Rate & Retention Rate
* Customer segmentation by tenure and income
* High-risk customer identification
* Multi-factor churn analysis

---

## 📈 Power BI Dashboard

Built an interactive dashboard including:

* KPIs (Churn Rate, Total Customers, Avg Tenure)
* Churn distribution and trends
* Customer segmentation visuals
* Interactive filters

---

## 🌐 Deployment

* Deployed interactive dashboard using Streamlit
* Hosted code on GitHub

---

## 💡 Key Insights

* Customers with low tenure have higher churn rates
* Service usage significantly impacts churn behavior
* Certain customer categories show higher churn risk
* Identified segments contributing most to churn

---

## 🚀 Business Recommendations

* Target new customers with retention offers
* Improve services for high-risk segments
* Provide personalized offers based on usage patterns
